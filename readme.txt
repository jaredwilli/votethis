== Description ==

This plugin allows your visitors to vote or like your posts.

= This plugin include =
* A counter/button to display the number of "like" and to vote.
* A widget and a function to display the X most liked posts.
* A preference pane with some options.

== Installation ==

1. Upload the directory `/like-this/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
